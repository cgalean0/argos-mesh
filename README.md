# argos-mesh
Distributed system to prevent attacks of DDOS toward a mini Shop

### Estructura de carpetas dentro del proyecto
/argos-sentinel (El protector con Java 21 y Virtual Threads).

/argos-orders (La lógica de la tiendita).

/argos-notify (El worker de notificaciones).

/infra (Donde pondremos los Dockerfiles y los archivos de Kubernetes).
